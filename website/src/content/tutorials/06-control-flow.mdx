# 控制流

## 你会学到什么

- `if` 是表达式，可以产生值。
- `loop` 可以用 `break value` 返回值。
- `for` 是遍历集合和 range 的常用方式。

## 最小示例

```rust
let level = if score >= 60 { "通过" } else { "继续练习" };

for chapter in 1..=3 {
    println!("第 {chapter} 章");
}
```

## 代码讲解

`if` 的两个分支必须返回同一种类型：

```rust
let status = if score >= 60 { "pass" } else { "retry" };
```

`for` 常用于遍历 range、array、slice、`Vec` 和任何实现了 `IntoIterator` 的类型。

`loop` 适合表达“直到满足条件才退出”的逻辑：

```rust
let mut value = 0;
let result = loop {
    value += 1;
    if value == 3 {
        break value * 10;
    }
};
```

## 常见错误

Rust 不会把数字自动当成布尔值：

```rust
let count = 1;
if count {
    println!("has value");
}
```

修复方式是写出明确条件：

```rust
if count > 0 {
    println!("has value");
}
```

## 练习

- 写一个函数，根据分数返回 `"A"`、`"B"`、`"C"` 或 `"D"`。
- 用 `for` 计算 `1..=100` 的和。

## 小结

Rust 的控制流强调类型一致和条件明确。分支和循环都可以参与表达式求值。
