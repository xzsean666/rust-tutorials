# 借用与引用

## 你会学到什么

- 用 `&` 创建共享借用，用 `&mut` 创建可变借用。
- 借用规则：要么多个共享借用，要么一个可变借用，不能同时。
- 借用不转移所有权，调用方仍然拥有数据。

## 最小示例

```rust
let mut message = String::from("hello");
append_exclaim(&mut message); // 可变借用
println!("{message}"); // message 仍然属于这里
```

## 代码讲解

共享借用 `&T` 是只读的，可以同时存在多个：

```rust
fn first_word(text: &str) -> &str {
    text.split_whitespace().next().unwrap_or("")
}
```

可变借用 `&mut T` 允许修改，但同一时刻只能有一个：

```rust
fn append_exclaim(text: &mut String) {
    text.push('!');
}
```

借用检查器在编译期保证：不会出现“一边读一边写”的数据竞争。

## 常见错误

同时持有可变借用和共享借用：

```rust
let mut v = vec![1, 2, 3];
let first = &v[0];   // 共享借用
v.push(4);           // ❌ 可变借用，与 first 冲突
println!("{first}");
```

修复方式是缩小借用范围，让 `first` 在 `push` 之前用完。

## 练习

- 写一个函数 `&mut Vec<i32>`，把每个元素翻倍。
- 写一个函数返回切片中的最大值引用 `&i32`。

## 小结

借用让你在不转移所有权的前提下访问数据。共享借用可读不可写且可多个，可变借用唯一且可写。
